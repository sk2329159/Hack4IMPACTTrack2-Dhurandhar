## Approved Problem Statement
Mitigating National Security Risks Posed by Large Language Models (LLMs) in AI-Driven Malign Information Operations

## Project Overview
SENTINEL-AI is an AI-native threat intelligence platform designed to detect, attribute, and map AI-generated malign information operations in real time. The platform combines transformer-based AI detection, stylometric attribution, and graph-based campaign analysis to support cybersecurity and national security use cases.

## Core Features
- AI-generated text detection
- Model attribution (GPT-family / Claude-family / Unknown)
- Risk scoring
- Campaign propagation graph
- Dashboard for analysis and monitoring
- Exportable threat reports

## Prototype Scope
This 36-hour prototype demonstrates:

- trend analytics
